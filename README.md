# AI OS Generator

A reusable project system for generating SaaS and AI products through structured skills, playbooks, and orchestration.

## Structure
- `CLAUDE.md` → global operating rules
- `orchestrator/` → routing and execution logic
- `skills/` → reusable specialized capabilities
- `templates/` → standardized output formats
- `playbooks/` → project-type specific flows
- `project-input/` → current project idea and constraints
- `outputs/` → generated artifacts

## Goal
Turn ideas into production-ready SaaS and AI systems with consistent structure, quality, and speed.
