# Phase Router

## Goal
Route the right skills based on the shape of the project.

## Routing Logic

### If project includes:
- chat
- assistant
- documents
- legal text
- knowledge base
Then prioritize:
- research
- product scope discipline
- architecture with retrieval readiness

### If project includes:
- subscription
- premium features
- paid plans
- upgrade path
Then prioritize:
- monetization clarity
- MVP gating
- pricing-aware launch strategy

### If project includes:
- dashboard
- admin
- analytics
- internal tooling
Then prioritize:
- dashboard structure
- role-aware data design
- operational clarity

### If project includes:
- forms
- workflows
- case submission
- incident collection
Then prioritize:
- entity design
- onboarding flow
- service layer planning

## Standard Route
1. Classify
2. Analyze market
3. Define MVP
4. Design architecture
5. Design schema
6. Design dashboard
7. Prepare launch strategy

## Constraint
Never skip classification and MVP definition.
