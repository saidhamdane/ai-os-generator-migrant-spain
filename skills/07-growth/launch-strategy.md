# Skill: Launch Strategy

## Goal
Define the fastest path to acquiring early users and first revenue.

## When To Use
After MVP scope and positioning are clear.

## Inputs
- market analysis output
- MVP definition
- `project-input/monetization.md`

## Steps
1. Define the core offer.
2. Define the target early-adopter audience.
3. Choose 1 to 3 launch channels.
4. Define a simple acquisition message.
5. Define conversion path from visitor to paid user.
6. Define one feedback loop after launch.

## Output
- launch positioning
- ideal early adopters
- recommended channels
- message angle
- conversion path
- first revenue plan
- feedback loop

## Constraints
- Avoid broad multi-channel launches in v1.
- Pick channels where the target user already pays attention.
- Keep the plan executable by a small team.

## Success Criteria
The project has a concrete first-market-entry plan instead of a vague “we will market later” approach.
