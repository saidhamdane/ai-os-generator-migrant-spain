# Constraints

- MVP must be simple, focused, and fast to launch
- Web-first product
- Mobile-friendly UI
- Arabic + Spanish initially
- Later add English and French
- Use Next.js + Tailwind + PostgreSQL + Prisma
- Stripe for subscriptions
- Resend for email
- Vercel for deployment
- Start with guidance, checklists, and document workflows before adding marketplace or lawyer network
